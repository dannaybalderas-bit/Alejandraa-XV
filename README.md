# Invitación XV Alejandra - Versión final

Archivos listos para GitHub Pages.

## Para publicar
1. Descomprime el ZIP.
2. Sube TODO el contenido de esta carpeta a tu repositorio.
3. Haz **Commit changes**.
4. Espera unos minutos a que GitHub Pages actualice.

## Incluye
- Vista previa para WhatsApp (`assets/img/preview.jpg`).
- Favicon e iconos de app (`assets/icons/`).
- Manifest para agregar a pantalla de inicio.
- Música, fotos, ubicación, dress code y sobre.

## Importante
La vista previa de WhatsApp puede tardar en actualizarse por caché. Si no cambia de inmediato, espera unos minutos o prueba pegar el enlace en otro chat.
